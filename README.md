# CodeAlpha_DiseasePrediction
# Disease Prediction from Medical Data

## Project Overview

This project uses Machine Learning algorithms to predict the likelihood of heart disease based on patient medical data.

## Dataset

Heart Disease Dataset

## Algorithms Used

- Logistic Regression
- Random Forest Classifier

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score

## Results

Random Forest achieved the best performance and was selected as the final model.
